Detailed Description

Class holding the state of the system.

Author:
Simon Walker

Structure that holds the parameters for the model system. Is taken as a parameter in the main model generation function. The parameters are given in some standard units which are explained with the description of the parameters. For example the separation is in AU (Earth-Sun distance).

Some of the parameters are not used directly in the lightcurve generation code, but are by some other codes I have written. The non-required parameters are:

  • id
  • name
  • submodel_id
  • mstar
  • teff

These can be set to whatever.

Limb darkening

The limb darkening is modeled using 4 parameters: \(c1\), \(c2\), \(c3\) and \(c4\). These are related to \(c0\) by

\[ c0 = 1 - c1 - c2 - c3 - c4 \]

These are calculated separately but must all be given for a realistic model.
